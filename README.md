## Features:
- Parses Cowrie honeypot `json` logs into `sqlite` at roughly `20,000
  insertions/sec`, adding geolocation data from MaxMind.
- Low memory consumption 

## Requirements
- `libgeos-dev`, `libgdal-dev`, `libproj-dev`
- `basemap`
